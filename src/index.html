Noix

Noix is a barebones template to kick start web projects. It comes with:

  • Grunt to automate tasks
  • Bower to install libraries
  • Responsive Grid for layouts
  • //todo: Styles for commonly used elements
  • //todo: A skeleton MVC template
  • //todo: Routing system
